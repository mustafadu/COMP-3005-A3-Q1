To run the code:
- open the code
- uupdate your "database", "postgres", "host", "password", "port" information
- open cmd
- by using cmd, go to the directory
- run python3 program.py
- the program will ask you 5 questions
- to print all the students in the database press 1
- to add a student press 2, and provide correct information
- to update a student' email press 3, provide correct information
- to delete a student press 4, and provide student' number
- to exit the program, press 5
